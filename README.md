# Cardiac MRI Simulator

## Overview

Developed a physics-inspired cardiac motion simulator to generate synthetic displacement fields and principal strain maps for validating deep learning methods in cardiac MRI.

## Motivation

Obtaining pixel-level ground truth motion in cardiac MRI is challenging. This simulator generates realistic synthetic cardiac deformation that can be used to develop, benchmark, and validate motion estimation algorithms.

## Features

- Synthetic cardiac motion generation
- Configurable deformation fields
- MRI image deformation
- Principal strain computation
- Visualization of displacement and strain maps
- Animation of simulated cardiac motion

## Technologies

- Python
- NumPy
- SciPy
- OpenCV
- Matplotlib

## Repository Status

This repository summarizes work completed as part of ongoing NIH research. The source code is not publicly available.
